AltspaceVR Projects

Thank you for applying to AltspaceVR! Part of our hiring process is having candidates complete a small programming project. Feel free to choose any of these projects:

Unity: 3D Cursor

https://github.com/AltspaceVR/altspacevr-project-unity-cursor

Finish the implementation of the 3D cursor in an example Unity application, and then build some enhancements using the cursor.

three.js: VR App

https://github.com/AltspaceVR/altspacevr-project-threejs

Create a small three.js/webgl application that would be interesting to see inside of VR.

HTML5 UI: VR App

https://github.com/AltspaceVR/altspacevr-project-html-ui

Create a small web application that demonstrates a HTML5 user interface that would be important or interesting in VR.

Operations: TodoMVC

https://github.com/AltspaceVR/altspacevr-project-todomvc-ops

Create a small virtualized/containerized environment to operationalize a TodoMVC app.

Rails: HMD Tracker

https://github.com/AltspaceVR/altspacevr-project-hmdtracker-rails

Update an existing Rails project to audit changes to a state table, and then include additional enhancements of your own.

We hope you have fun working on the project, and we can't wait to see what you come up with!
